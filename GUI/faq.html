    • How Tokenizing is done in this project ?

      Java code was used to normailze the text and convert it to tokenstream. XML file was read via SAX parser and Wiki parsers extracted the text from the XML dump. This data was then fed in a HashMap one document at a time and then index into Solr. Re-direct pages were omitted.

    • How Indexing is implemented in this project ?

      Native Solr features were used to convert the incoming Tokenstream from SolrJ to index. We used the standard tokenizer factory to remove special charatcters. Solr Filters like punctuation were used to get rid of characthers and whitespace normailzers. The modified stream is then indexed.

    • Query Processing in this project ?

      The query is captured by the HTML form is java which does an HTTP POST to the jsp file in Tomcat which then gathers the input. JSP file generates a Solr query and renders the incoming JSON into a user readable neatly formated page. This shows the results for each core separately.
